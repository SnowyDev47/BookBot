# 🤖📕 BookBot 📕🤖

***This is my first project ever made!***

This script grabs a .txt file and prints the number of words and then lists the count of each character.
You can put the .txt file in the books folder for convenience.

## 💻 Usage 💻

**python3 main.py <path_to_book>**
